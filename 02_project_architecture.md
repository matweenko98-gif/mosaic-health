# Архитектура проекта и структура папок

Для реализации прототипа из двух экранов рекомендуется следующая структура файлов в проекте Antigravity. Она позволит легко масштабировать приложение до полноценного MVP в будущем.

## Рекомендуемое дерево каталогов (src/)

```text
src/
├── components/               # Переиспользуемые логические блоки интерфейса
│   ├── BottomNav.jsx         # Нижняя панель навигации (переключатель экранов)
│   ├── BentoGrid.jsx         # Модульная сетка блоков для Главного экрана
│   ├── WorkoutModal.jsx      # Модальное окно "плеера" тренировки
│   └── ProfileForm.jsx       # Интерактивная форма личных данных пользователя
├── screens/                  # Основные экраны приложения
│   ├── HomeScreen.jsx        # Экран: Главная страница
│   └── ProfileScreen.jsx     # Экран: Личный кабинет / Профиль
├── data/                     # Статические и начальные данные
│   └── mockData.js           # Тексты, начальная история и настройки
├── App.jsx                   # Корневой компонент (управление глобальным State)
├── main.jsx                  # Точка входа в React-приложение
└── index.css                 # Базовые стили прототипа
